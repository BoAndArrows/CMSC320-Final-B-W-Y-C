North American Bird Banding Program Dataset 1960-2023

This readme describes file type, field names and all bird species in each data file for download. 

57 data files NABBP_2023_grp_01 through NABBP_2023_grp_57 are comma-separated values (.csv) files with header, UTF8 encoded. Each file contains banding, encounter and recapture data for bird species based on taxonomic groups. Mallard (1320-MALL, Anas platyrhynchos) is in a separate file (group 03) with 7.7 million records. These large .csv files may be viewed in applications such as R, Windows Powershell, SQLite DB Browser. Use species_id field to relate data file with provided species lookup table.

Each file includes 29 fields:
record_id, event type, band, original band, other bands, event date, event day, event month, event year, species id, species_name, ISO country, ISO subdivision, latitude (decimal degrees), longitude (decimal degrees), coordinate precision code, age code, sex code, bird status, extra info code, how obtained code, who obtained code, reporting method, present condition, minimum age at encounter, permit, band status code, band type code, record source.

Original band numbers and permit numbers are obfuscated. For alphanumeric codes interpretation, download the zipped file of lookup tables associated with this 2023 dataset.

Below are groups 01-57 with general descriptions to help select data file(s) for download. If you don't find the group you are looking for, scroll down for a detailed list by species.
